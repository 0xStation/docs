GroupOS's API enables seamless integration between on-chain transactions and off-chain logic and events. The developer console is the central hub for developers to track API usage and status.

**Key Features**

<CardGroup cols={1}>
  <Card title="Organization's API Key" icon="square-1" color="#ad72ff">
    Securely view and manage your organization's unique API key. 
  </Card>
  <Card title="Gas Usage Metrics" icon="square-2" color="#ad72ff">
    Monitor total and per-transaction gas usage for insights into efficiency and costs.
  </Card>
  <Card title="API Requests" icon="square-3" color="#ad72ff">
    Track successful and failed API requests for proactive monitoring and quick issue response.
  </Card>
  <Card title="API Logs" icon="square-4" color="#ad72ff">
    Filter and sort relevant API logs for efficient tracking and debugging.
  </Card>
</CardGroup>


**Example**

GroupOS's API can integrate with gaming environments, allowing developers to create new on-chain accounts when users complete in-game actions.

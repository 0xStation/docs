---
title: "Query ERC-1155 Token Owners"
sidebarTitle: "Query Token Owners"
api: "GET https://groupos.xyz/api/v1/erc1155/tokenOwners"
---

### Query Params

<ParamField query="chainId" type="number" required>
  The id of the chain supporting the NFT.
</ParamField>
<ParamField query="contractAddress" type="string" required>
  The NFT smart contract address.
</ParamField>
<ParamField query="ownerAddress" type="string" optional>
  An NFT owner to filter on.
</ParamField>
<ParamField query="tokenId" type="string" optional>
  An NFT `tokenId` to filter on.
</ParamField>
<ParamField query="page" type="number" optional>
  Page index to fetch, page size is `50`.
</ParamField>

### Response

<ResponseField name="success" type="boolean">
  Indicates whether the call was successful.
</ResponseField>
<ResponseField name="tokenOwners" type="object[]">
  Array of NFT owner objects.
  <Expandable title="object">
    <ResponseField name="tokenId" type="string">
      Token id of the NFT.
    </ResponseField>
    <ResponseField name="ownerAddress" type="string">
      Address of an owner (lower-cased).
    </ResponseField>
    <ResponseField name="amount" type="string">
      The amount of tokens of `tokenId` owned by `ownerAddress`.
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
  ```bash Example Request
  curl --location 'https://groupos.xyz/api/v1/erc1155/tokenOwners' \
  --request GET \
  -d 'chainId=1' \
  -d 'contractAddress=0x67F4732266C7300cca593C814d46bee72e40659F' \
  -d 'ownerAddress=0xc517c83f417b73dA98647dad0FCB80af9f3b9531' \
  -d 'tokenId=69' \
  --header 'Content-Type: application/json' \
  --header 'Authorization: Bearer <token>'
  ```
</RequestExample>

<ResponseExample>
  ```json Success
  {
    "success": true,
    "tokenOwners": [
      {
        "tokenId": "69",
        "ownerAddress": "0xc517c83f417b73da98647dad0fcb80af9f3b9531",
        "amount": "12",
      }
      ...
    ]
  }
  ```
  ```json Failure
  {
    "success": false,
    "error": "[message]"
  }
  ```
</ResponseExample>

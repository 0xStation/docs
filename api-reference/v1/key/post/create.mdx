### Key custody

Currently, keys are custodied in the cloud with secure enclave technology through our partner [Turnkey](https://www.turnkey.com/).
This allows you and your users to easily onboard to a dApp without needing to touch private keys. For more information about
key security and operations, please contact us at our Help Desk.

### Body

<ParamField body="customRef" type="string" required={false}>
  A custom reference to associate with the key for later querying. Can be
  anything you want to identify keys or users by, eg. a user's email or a
  database UUID.
</ParamField>

### Response

<ResponseField name="success" type="boolean">
  Indicates whether the call was successful.
</ResponseField>

<ResponseField name="evmAddress" type="string">
  Public EVM Address for the new key.
</ResponseField>

<RequestExample>

```bash Example Request
curl --location --request POST 'https://groupos.xyz/api/v1/key/create' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer <token>' \
--data-raw '{
    "customRef": "user@email.com"
}'
```

</RequestExample>

---

### Query Params

<ParamField query="customRef" type="string" optional>
  The custom reference associated with the key at creation.
</ParamField>
<ParamField query="page" type="number" optional>
  Page index to fetch, page size is `50`.
</ParamField>

### Response

<ResponseField name="success" type="boolean">
  Indicates whether the call was successful.
</ResponseField>
<ResponseField name="keys" type="object[]">
  Array of key objects.
  <Expandable title="object">
    <ResponseField name="evmAddress" type="string">
      Public EVM Address for the key.
    </ResponseField>
    <ResponseField name="keyId" type="string">
      The ID associated with the key at creation.
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
  ```bash Example Request
  curl --location 'https://groupos.xyz/api/v1/key/keys' \
  --request GET \
  -d 'customRef=user@email.com'
  --header 'Content-Type: application/json' \
  --header 'Authorization: Bearer <token>'
  ```
</RequestExample>

<ResponseExample>
  ```json Success
  {
    "success": true,
    "keys": [
      {
        "evmAddress": "0xc517c83f417b73da98647dad0fcb80af9f3b9531",
        "customRef": "user@email.com",
      }
    ]
  }
  ```
  ```json Failure
  {
    "success": false,
    "error": "[message]"
  }
  ```
</ResponseExample>

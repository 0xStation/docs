---
title: "Update NFT Metadata"
---

Update NFT Metadata based on set conditions to reflect the evolving identity of your members. 

**Examples**
- If member has donated, update the metadata from 'member' to 'donor'.
- If member hasn't logged into the app for more than 60 days, update the metadata from 'active' to 'inactive'

### Set up API Keys

You can skip this section if you've already set up in the API keys.

1. In the Dashboard, go to the 'Settings' tab. 
2. Go to the 'API keys' and click 'Add' to generate a new key.
![API.png](https://station-images.nyc3.digitaloceanspaces.com/8e6e553e-1da4-4501-ac64-52094d9c1a0d.png)
3. Name your API key and click 'Generate'.

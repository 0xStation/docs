## Functions
### pointsImpl

*Get the current implementation contract*


```solidity
function pointsImpl() external view returns (address);
```

### setPointsImpl

*Set the `newImpl` address whose logic will be used by deployed Points proxies*


```solidity
function setPointsImpl(address newImpl) external;
```

### initialize

*Initializes the proxy for the factory*


```solidity
function initialize(address pointsImpl_, address owner_) external;
```
**Parameters**

|Name|Type|Description|
|----|----|-----------|
|`pointsImpl_`|`address`|The initial Points implementation address whose logic to use|
|`owner_`|`address`|The owner address to be set for the factory contract|


### create

*Function to create a new Points proxy using given data*


```solidity
function create(address owner, string memory name, string memory symbol, bytes calldata initData)
    external
    returns (address points);
```
**Parameters**

|Name|Type|Description|
|----|----|-----------|
|`owner`|`address`|The owner address to be set for the new Points|
|`name`|`string`|The points name string|
|`symbol`|`string`|The points symbol string|
|`initData`|`bytes`|Data to pass to `initialize()` on the created Points proxy|

**Returns**

|Name|Type|Description|
|----|----|-----------|
|`points`|`address`|The created Points proxy address|

**View API logs**

Station's API enables seamless integration between on-chain transactions and off-chain logic and events. The developer console is the central hub for developers to track API usage and status.

**Key Features**

- **Organization's API Key:** Securely view and manage your organization's unique API key.
- **Gas Usage Metrics:** Monitor total and per-transaction gas usage for insights into efficiency and costs.
- **API Requests:** Track successful and failed API requests for proactive monitoring and quick issue response.
- **API Logs:** Filter and sort relevant API logs for efficient tracking and debugging.

**Example**

Station's API can integrate with gaming environments, allowing developers to create new on-chain accounts when users complete in-game actions.

### Query Params

<ParamField query="chainId" type="number" required>
  The id of the chain supporting the token. This cannot be changed after creation.
</ParamField>
<ParamField query="contractAddress" type="string" required>
  The token smart contract address.
</ParamField>
<ParamField query="ownerAddress" type="string" optional>
  An token owner to filter on.
</ParamField>
<ParamField query="page" type="number" optional>
  Page index to fetch, page size is `50`.
</ParamField>

### Response

<ResponseField name="success" type="boolean">
  Indicates whether the call was successful.
</ResponseField>
<ResponseField name="tokens" type="TokenOwner[]">
  Array of TokenOwner objects.
  <Expandable title="TokenOwner">
    <ResponseField name="ownerAddress" type="string">
      Address of an owner (lower-cased).
    </ResponseField>
    <ResponseField name="amount" type="string">
      The amount of tokens owned by `ownerAddress`.
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
  ```bash Example Request
  curl --location 'https://groupos.xyz/api/v1/erc20/tokens' \
  --request GET \
  -d 'chainId=1' \
  -d 'contractAddress=0x67F4732266C7300cca593C814d46bee72e40659F' \
  -d 'ownerAddress=0xc517c83f417b73dA98647dad0FCB80af9f3b9531' \
  --header 'Content-Type: application/json' \
  --header 'Authorization: Bearer <token>'
  ```
</RequestExample>

<ResponseExample>
  ```json Success
  {
    "success": true,
    "tokens": [
      {
        "ownerAddress": "0xc517c83f417b73da98647dad0fcb80af9f3b9531",
        "amount": "12",
      }
    ]
  }
  ```
  ```json Failure
  {
    "success": false,
    "error": "[message]"
  }
  ```
</ResponseExample>

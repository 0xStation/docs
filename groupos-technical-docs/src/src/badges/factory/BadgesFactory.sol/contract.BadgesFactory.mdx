### initialize

*Initializes the proxy for the factory*


```solidity
function initialize(address badgesImpl_, address owner_) external initializer;
```
**Parameters**

|Name|Type|Description|
|----|----|-----------|
|`badgesImpl_`|`address`|The initial Badge implementation address whose logic to use|
|`owner_`|`address`|The owner address to be set for the factory contract|

### create

*Function to create a new Badge proxy using given data*


```solidity
function create(address owner, string memory name, string memory symbol, bytes calldata initData)
    public
    returns (address badges);
```
**Parameters**

|Name|Type|Description|
|----|----|-----------|
|`owner`|`address`|The owner address to be set for the new Badge|
|`name`|`string`|The badge name string|
|`symbol`|`string`|The badge symbol string|
|`initData`|`bytes`|Data to pass to `initialize()` on the created Badge proxy|

**Returns**

|Name|Type|Description|
|----|----|-----------|
|`badges`|`address`|The created Badge proxy address|

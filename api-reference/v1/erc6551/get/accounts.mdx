---
title: "Query NFT Accounts"
sidebarTitle: "Query Accounts"
api: "GET https://groupos.xyz/api/v1/erc6551/accounts"
---

Returns accounts associated with your Group which includes created and imported ERC-721 NFTs.

### Query Params

<ParamField query="page" type="number" optional>
  Page index to fetch, page size is `50`.
</ParamField>

### Response

<ResponseField name="success" type="boolean">
  Indicates whether the call was successful.
</ResponseField>

<ResponseField name="accounts" type="object[]">
  Array of NFT Account objects.

  <Expandable title="object">

      <ResponseField name="contractAddress" type="string">
        Address of the Account.
      </ResponseField>

      <ResponseField name="implementationAddress" type="string">
        Address of the Account's implementation.
      </ResponseField>

      <ResponseField name="nft" type="object">
        NFT this Account is bound to.
        <Expandable title="object">
          <ResponseField name="chainId" type="string">
            Chain ID of where this NFT is deployed on.
          </ResponseField>

          <ResponseField name="contractAddress" type="string">
            Contract address of the NFT collection.
          </ResponseField>

          <ResponseField name="tokenId" type="string">
            Token ID of the NFT within the collection.
          </ResponseField>
        </Expandable>
      </ResponseField>

  </Expandable>

</ResponseField>

<RequestExample>

```bash Example Request
curl --location 'https://groupos.xyz/api/v1/erc6551/accounts' \
--request GET \
-d 'accountGroupId=6e4adbcf-ef53-49aa-82c0-2e10ae4b0d84' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer <token>'
```

</RequestExample>

## Set up your API key

1. Sign into [GroupOS](https://groupos.xyz/). 
2. In the Dashboard, go to the **Settings** tab.
3. Go to the **API keys** tab and click 'Add' to generate a new key.
![API.png](https://station-images.nyc3.digitaloceanspaces.com/8e6e553e-1da4-4501-ac64-52094d9c1a0d.png)
4. Name your API key and click **'Generate'**. 
<Warning>Save the key in a secured location to reuse the key.</Warning>

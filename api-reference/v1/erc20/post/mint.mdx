### How to mint ERC-20 tokens

Once you have a ERC-20 Token smart contract, you can mint any number of tokens to any address.

### Body

<ParamField body="chainId" type="number" required>
  The id of the chain supporting the token. This cannot be changed after
  creation.
</ParamField>

<ParamField body="contractAddress" type="string" required>
  The token smart contract address to mint with.
</ParamField>

<ParamField body="recipientAddress" type="string" required>
  The address to mint an token to.
</ParamField>

<ParamField body="amount" type="string" required>
  The human-readable quantity of points to mint. Type `string` is used to
  natively support large numbers.
</ParamField>

### Response

<ResponseField name="success" type="boolean">
  Indicates whether the call was successful.
</ResponseField>

<ResponseField name="transactionHash" type="string">
  Transaction hash for the newly created token.
</ResponseField>

<ResponseField name="transactionUrl" type="string">
  Wraps `transactionHash` with the appropriate network-specific URL for
  convenience.
</ResponseField>

<RequestExample>

```bash Example Request
curl --location --request POST 'https://groupos.xyz/api/v1/erc1155/mint' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer <token>' \
--data-raw '{
    "chainId": 137,
    "contractAddress": "0x67F4732266C7300cca593C814d46bee72e40659F",
    "recipientAddress": "0xc517c83f417b73dA98647dad0FCB80af9f3b9531",
    "amount": "420",
}'
```

</RequestExample>

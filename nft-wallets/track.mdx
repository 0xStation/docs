## Wallet Directory

You can track wallets' activities, token ownership, and gas usage via Wallet Directory.

1. In the Group Dashboard, go to the Wallets tab.
2. Select the wallet you wish to view more information about.

**Coming soon**

- Analytics to track growth
- Mint and burn wallets from the Directory

## Developer Logs

Monitor the distribution and updating of the NFT Wallets via Logs.

1. In the Dashboard, go to **Logs**.
2. Filter by the wallet on the side bar.
